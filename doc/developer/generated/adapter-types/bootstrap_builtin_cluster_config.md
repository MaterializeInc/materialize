# mz-adapter-types::bootstrap_builtin_cluster_config

Defines `BootstrapBuiltinClusterConfig` (a size string and replication factor) and default replication factor constants for each system cluster: `system`, `catalog-server`, `probe` (factor 1), and `support`/`analytics` (factor 0, ephemeral).
