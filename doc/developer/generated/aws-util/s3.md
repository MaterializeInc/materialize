# mz-aws-util::s3

Provides S3 client construction and utilities for interacting with Amazon S3 (and S3-compatible endpoints).
Exposes `new_client`, which builds an `aws_sdk_s3::Client` with Materialize-specific customizations (path-style addressing when a custom endpoint is configured), `list_bucket_path` for listing objects under a prefix, and `ByteStreamAdapter`, which wraps `aws_smithy_types::byte_stream::ByteStream` to implement the `futures::stream::Stream` trait.
This module is only compiled when the `s3` feature is enabled.
