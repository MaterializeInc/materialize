# catalog::durable::upgrade::v80_to_v81

Migration from catalog version 80 to 81.
Adds the `auto_provision_source` field to `RoleAttributes`. For cloud environments (detected by checking the `mz_system` cluster replication factor and absence of `enable_password_auth`), existing roles whose names look like email addresses are assumed to have been auto-provisioned via Frontegg. Self-managed environments get no migration actions since the field defaults to `None`.
