# catalog::durable::upgrade::v81_to_v82

Migration from catalog version 81 to 82.
Validates that no continual task items or comments on continual tasks remain in the catalog snapshot. If any are found, the migration panics with a message instructing the operator to drop all continual tasks before upgrading. When no continual tasks are present, the migration produces no `MigrationAction`s and the snapshot passes through unchanged.
