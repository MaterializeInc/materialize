# catalog::durable::upgrade::v84_to_v85

Migration from catalog version 84 to 85.
A no-op migration: all types are JSON-compatible between v84 and v85, so no `MigrationAction`s are emitted. The only schema change at v85 is the addition of `AlterAddColumnV1` and `AlterSourceTimestampIntervalV1` audit log event detail structs to `objects_v85`.
