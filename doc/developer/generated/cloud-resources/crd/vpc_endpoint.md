# cloud-resources::crd::vpc_endpoint

Defines the `VpcEndpoint` Kubernetes custom resource (group `materialize.cloud`, version `v1`) that instructs the environment-controller to provision an AWS VPC endpoint, along with `VpcEndpointSpec`, `VpcEndpointStatus`, and `VpcEndpointState`.
