# mz-expr::scalar::func::impls::byte

Provides scalar function implementations for `bytea` datums: casts to/from string representations, `length`, `bit_length`, `octet_length`, `substr`, `ltrim`, `rtrim`, `btrim`, `encode`/`decode`, and `get_byte`/`set_byte`.
