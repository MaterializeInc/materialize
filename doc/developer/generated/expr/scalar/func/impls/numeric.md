# mz-expr::scalar::func::impls::numeric

Provides scalar function implementations for the arbitrary-precision `numeric` type: arithmetic, rounding, square root, natural/base-10/base-2 logarithm, power, casts to/from all other numeric types and string, and special-value checks.
