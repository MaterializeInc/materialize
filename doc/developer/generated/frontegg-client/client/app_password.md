# frontegg-client::client::app_password

Implements `list_app_passwords` and `create_app_password` methods on `Client`, providing typed access to the Frontegg app-password API.
Defines `AppPassword` (metadata: description, created_at) and `CreateAppPasswordRequest` as the request/response types for these operations.
