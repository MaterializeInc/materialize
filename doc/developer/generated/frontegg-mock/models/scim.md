# frontegg-mock::models::scim

Defines `SCIM2ConfigurationCreateRequest`, `SCIM2ConfigurationResponse`, and `SCIM2ConfigurationStorage` for SCIM 2.0 provisioning configuration objects.
`SCIM2ConfigurationStorage` is the server-side in-memory representation (including a generated bearer token), while `SCIM2ConfigurationResponse` is the serialized API response.
