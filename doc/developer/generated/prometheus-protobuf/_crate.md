# `prometheus-protobuf`

A thin helper crate for decoding the Prometheus protobuf scrape format. The crate is isolated from the rest of the codebase because the `protobuf` crate it depends on is banned from general use (it is unmaintained), but it is used here out of convenience since `prometheus::proto::MetricFamily` is already generated by `protobuf`.

## Public API

### `decode_length_delimited`

```rust
pub fn decode_length_delimited(bytes: &[u8]) -> Result<Vec<proto::MetricFamily>, String>
```

Decodes the length-delimited protobuf wire format produced by [`prometheus::ProtobufEncoder`] — the format callers receive from a Prometheus `/metrics` endpoint with `Accept: application/vnd.google.protobuf`. Returns a `Vec<proto::MetricFamily>` on success, or an error string describing the first failure.

The function also re-exports `prometheus::proto` so callers can work with `proto::MetricFamily` without a direct dependency on the `prometheus` crate.
