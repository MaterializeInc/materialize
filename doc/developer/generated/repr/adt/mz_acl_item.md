# mz-repr::adt::mz_acl_item

Defines `MzAclItem` (Materialize-specific ACL entry with `GlobalId`-based grantee/grantor) and `AclItem` (PostgreSQL-compatible OID-based ACL entry), along with `AclMode` bitflags for privilege representation.
