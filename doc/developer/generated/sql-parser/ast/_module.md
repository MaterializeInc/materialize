# mz-sql-parser::ast

The SQL Abstract Syntax Tree module, re-exporting all AST types from `defs` and `metadata` as a flat namespace.
`display` provides the `AstDisplay` trait and rendering infrastructure; `fold`, `visit`, and `visit_mut` provide ownership-transforming and (im)mutable traversal traits, all auto-generated by the build script.
Consumers import from `mz_sql_parser::ast` directly without needing to track which sub-module a type lives in.
