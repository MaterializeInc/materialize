# mz-sql-parser::ast::fold

Provides the auto-generated `Fold` trait for ownership-transforming traversal of the AST, mapping `AST<T>` to `AST<T2>` for different `AstInfo` instantiations.
Each method has a default implementation that recursively folds child nodes; callers override specific methods to inject type-resolution logic (e.g., resolving raw names to augmented names during planning).
This file is generated by the `mz-walkabout` build script from the AST type definitions.
