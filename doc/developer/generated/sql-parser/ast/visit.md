# mz-sql-parser::ast::visit

Provides the auto-generated `Visit` trait for immutable traversal of the AST.
Each method visits a specific node type and by default recurses into all child nodes; callers override individual methods to observe or collect information from nodes of interest without modifying the tree.
Generated by `mz-walkabout` from the AST type definitions.
