# mz-sql-parser::ast::visit_mut

Provides the auto-generated `VisitMut` trait for mutable traversal of the AST, analogous to `Visit` but operating on `&mut` references.
Callers override individual methods to rewrite nodes in place, enabling transformations such as identifier normalization or desugaring.
Generated by `mz-walkabout` from the AST type definitions.
