# testdrive::action::s3

Implements the `s3-put-object` builtin command for uploading objects to S3 (or a compatible store).
Supports plain text, CSV, and Parquet formats; for Parquet it builds a full Arrow schema from a column-type specification and writes a `RecordBatch`.
For non-Parquet formats the content may be additionally compressed (bzip2, gzip, xz, zstd) before upload.
