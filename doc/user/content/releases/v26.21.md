## v26.21.0

#### SQL

* Add the `mz_ontology` system schema containing four new catalog ontology
  tables: `mz_ontology_entity_types`, `mz_ontology_semantic_types`,
  `mz_ontology_properties`, and `mz_ontology_link_types`. These tables describe
  the structure and relationships of the Materialize system catalog, including
  entity types, semantic column types (e.g., `CatalogItemId` vs `GlobalId`),
  column metadata, and named relationships between catalog tables. This schema
  is currently unstable and requires
  `ALTER SYSTEM SET unsafe_enable_unstable_dependencies = true` to use.
