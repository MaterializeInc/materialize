`v1alpha1` is the default CRD version (used before v26.30 and still the default).
Starting in v26.30, the [v1](/self-managed-deployments/materialize-crd-field-descriptions/v1/)
CRD is available; to get its automatic-rollout behavior, see [Adopting the v1
CRD](/self-managed-deployments/upgrading/#adopting-the-v1-crd).
{{</ note >}}
