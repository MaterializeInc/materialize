{{< show-command-note >}}

`SHOW DATABASES` returns a list of all databases available to your Materialize
instances.

## Syntax

{{< diagram "show-databases.svg" >}}

## Details

### Output format

`SHOW DATABASES`'s output is a table with one column, `name`.

## Examples

```sql
CREATE DATABASE my_db;
```
```sql
SHOW DATABASES;
```
```nofmt
materialize
my_db
```
