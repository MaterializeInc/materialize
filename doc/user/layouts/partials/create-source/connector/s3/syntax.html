**DISCOVER OBJECTS** | Describes how to discover keys to download. See [Object discovery strategies](#object-discovery-strategies). **BUCKET SCAN** *bucket_name* | Materialize will scan the bucket to find the set of objects to download. **SQS NOTIFICATIONS** *queue_name* | Materialize will subscribe to the specified queue and download new objects. See [Listening to SQS notifications](#listening-to-sqs-notifications). **MATCHING** *pattern* | A glob-style pattern to filter objects to ingest. See [Patterns](#patterns). Default is to ingest all objects. **COMPRESSION** _algorithm_ | The compression algorithm used to decode downloaded objects. See [Compression](#compression).
