The tutorial uses the `main.tf` found in the `examples/simple/` directory, which requires minimal user input. For details on the `examples/simple/` infrastructure configuration (such as the node instance type, etc.), see the [examples/simple/main.tf](https://github.com/MaterializeInc/terraform-azurerm-materialize/blob/main/examples/simple/main.tf). For more configuration options, you can run the `main.tf` file at the [root of the repository](https://github.com/MaterializeInc/terraform-azurerm-materialize/) instead. When running with the root `main.tf`, see [Azure required configuration](/installation/install-on-azure/legacy-terraform-module/appendix-configuration/).
