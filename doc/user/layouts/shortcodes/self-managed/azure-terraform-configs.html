The tutorial uses the `main.tf` found in the `examples/simple/` directory, which requires minimal user input. For details on the `examples/simple/` infrastructure configuration (such as the node instance type, etc.), see the [examples/simple/main.tf](https://github.com/MaterializeInc/terraform-azurerm-materialize/blob/main/examples/simple/main.tf). For more configuration options, you can run the `main.tf` file at the [root of the repository](https://github.com/MaterializeInc/terraform-azurerm-materialize/) instead. When running with the root `main.tf`: - Starting in v0.2.0, you must define the required providers. See [Providers Configuration](/installation/install-on-azure/appendix-azure-provider-configuration/) for details. - Starting in v0.2.0, you must specify the [network_config](https://github.com/MaterializeInc/terraform-azurerm-materialize?tab=readme-ov-file#input_network_config). In previous versions, a default value was provided.
