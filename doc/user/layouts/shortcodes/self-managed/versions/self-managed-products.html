For self-managed deployments, Materialize provides the Materialize Operator. The Materialize Operator manages Materialize environments within your Kubernetes infrastructure. Materialize provides the Materialize Operator Helm Chart, a [Helm Chart](https://helm.sh/docs/topics/charts/) for installing and upgrading the Materialize Operator. The [chart repository](https://helm.sh/docs/topics/chart_repository/) is: `https://materializeinc.github.io/materialize`.
