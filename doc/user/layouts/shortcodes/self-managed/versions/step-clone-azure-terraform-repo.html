For example, - If cloning via SSH (substitute `YOUR_ORGANIZATION` with your organization's name): ```bash git clone --depth 1 -b v0.2.0 git@github.com:YOUR_ORGANIZATION/terraform-azurerm-materialize.git ``` - If cloning via HTTPS (substitute `YOUR_ORGANIZATION` with your organization's name): ```bash git clone --depth 1 -b v0.2.0 https://github.com/YOUR_ORGANIZATION/terraform-azurerm-materialize.git ```
