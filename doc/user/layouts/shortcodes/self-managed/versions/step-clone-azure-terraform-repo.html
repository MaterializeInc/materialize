1. Fork the [Materialize's sample Terraform repo](https://github.com/MaterializeInc/terraform-azurerm-materialize). 1. Clone your forked repo and checkout the `v0.1.4` tag. For example, - If cloning via SSH (substitute `your-organization` with your organization's name): ```bash git clone --depth 1 -b v0.1.4 git@github.com:your-organization/terraform-azurerm-materialize.git ``` - If cloning via HTTPS (substitute `your-organization` with your organization's name): ```bash git clone --depth 1 -b v0.1.4 https://github.com/your-organization/terraform-azurerm-materialize.git ```
